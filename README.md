# Docker Compose Lab with CI/CD

![CI Status](https://github.com/umilomarr/Laba2/workflows/CI%20Pipeline/badge.svg)

## Описание проекта

Этот проект демонстрирует:
- Multi-container приложение с Docker Compose
- ASP.NET Core Web API + PostgreSQL
- Multi-stage Docker сборку
- GitHub Actions CI/CD pipeline
- Проверку качества с Hadolint
